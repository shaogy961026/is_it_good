新楓之谷角色 API 全資料擷取器

網站用途：
本工具用於快速擷取與檢視《新楓之谷》官方 API 實際開放的角色原始數據（JSON 格式）。輸入 API Key 後，即可一鍵獲取官方允許查詢的能力值、裝備、技能與戰地等底層資料，方便進行數據檢閱或開發，也適合單純好奇的你！
如何取得 API Key？
1. 前往 NEXON Open API 官方網站 並登入您的帳號。
2. 點擊右上角的 My Applications ➔ Register Application 註冊一個新應用程式。
3. 選擇「Maplestory (TW)」，填寫簡單名稱後送出。
4. 複製配發給您的 API Key（發行金鑰），貼到下方的欄位中即可使用。
* 您的金鑰僅會保存在您的瀏覽器本地，不會傳送給任何人，請安心使用。
  • 新楓之谷遊戲資料平均在 15 分鐘後即可使用。
  • 您可以從 2025 年 10 月 15 日起搜尋資料。
  • 您可以輸入所需日期以搜尋過往資料。前一日的資料將於翌日凌晨 2:00起提供。(當您搜尋 10 月 15 日的資料時，將會擷取從 15 日 00:00 到 16 日 00:00 的資料。)
  • 由於遊戲內容變動，OCID 可能會有所變更。（註：本工具每次皆會透過角色名稱即時換取最新 OCID，因此不受此變更影響，請安心使用！）
  • 此 API 提供來自新楓之谷台灣的資料。
